TANISHA ETHNICS — WEBSITE VIDEO UPLOAD

1. Instagram Reel ko MP4 format me download/export karein.
2. File ka simple name rakhein, example:
   new-arrival-01.mp4
   styling-reel-02.mp4
3. MP4 file isi videos folder me upload karein.
4. videos.json open karke entries add karein. Example:

[
  {
    "file": "new-arrival-01.mp4",
    "title": "New Arrival Collection",
    "caption": "Premium ethnic styles now available.",
    "category": "NEW ARRIVAL",
    "poster": "../brand-round.png"
  },
  {
    "file": "styling-reel-02.mp4",
    "title": "3 Ways to Style",
    "caption": "Elegant looks for festive occasions.",
    "category": "STYLE GUIDE"
  }
]

IMPORTANT:
- videos.json valid JSON hona chahiye.
- File name bilkul MP4 ke same hona chahiye.
- Recommended size: 1080x1920 vertical, under 15 MB per video.
- GitHub commit ke baad Netlify automatically deploy karega.
